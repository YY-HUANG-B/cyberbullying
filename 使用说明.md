# 心理学对比实验平台 - 使用说明

## 平台简介

这是一个基于Streamlit的心理学对比实验平台，专门用于网络欺凌干预研究。平台模拟了三个角色（欺凌者、受害者、治疗师）的对话交互，通过对比不同干预策略的效果，研究其对欺凌者攻击意图的影响。

## 快速启动

### 方法一：双击启动（推荐）
1. 双击 `启动应用.bat` 文件
2. 等待依赖检查完成
3. 浏览器将自动打开应用界面
4. 如果浏览器未自动打开，请手动访问：http://localhost:8502

### 方法二：使用Python脚本
1. 运行命令：`python launch_app.py`
2. 脚本会自动检查依赖并启动应用

### 方法三：手动启动
1. 确保已安装Python 3.8+
2. 安装依赖：`pip install -r requirements.txt`
3. 启动应用：`streamlit run app.py`

## 配置说明

### 1. API密钥配置
应用需要DeepSeek API密钥才能正常运行：
1. 访问 https://platform.deepseek.com/ 注册并获取API密钥
2. 编辑 `.env` 文件，将 `DEEPSEEK_API_KEY` 设置为您的密钥
3. 如果没有 `.env` 文件，启动脚本会自动从 `.env.example` 创建

### 2. 实验配置
在应用侧边栏可以配置：
- **讨论话题**：设置智能体讨论的话题
- **干预模式**：选择不同的干预策略
  - 模式A：纯共情干预
  - 模式B：纯理性规则干预
  - 模式C：高级整合干预（实验组）
- **API密钥**：实时更新API密钥

## 实验流程

### 1. 开始实验
1. 在侧边栏点击"开始/继续实验"按钮
2. 点击主界面的"开始下一轮对话"按钮
3. 系统会自动生成三轮对话：
   - 第一轮：Bully（欺凌者）发言
   - 第二轮：Victim（受害者）发言
   - 第三轮：Therapist（治疗师）干预

### 2. 数据监控
- **攻击性评分趋势图**：显示Bully攻击性分数的变化
- **对话历史**：显示完整的对话记录
- **内心独白**：显示Bully的心理活动变化

### 3. 数据管理
- **导出实验数据**：将对话数据导出为CSV文件
- **查看数据统计**：显示实验统计数据
- **重置实验**：清空当前实验数据

## 实验设计

### 自变量（干预模式）
- **模式A（控制组1）**：纯共情干预，无条件接纳情绪
- **模式B（控制组2）**：纯理性规则干预，强调法律后果
- **模式C（实验组）**：高级整合干预，结合多种心理技术

### 因变量（测量指标）
- **主要指标**：Bully的攻击性评分（0-10分）
- **次要指标**：Bully的内心独白（心理过程变化）
- **过程指标**：Victim的情绪转变轨迹

## 常见问题

### Q1: 应用无法启动
- 检查Python是否已安装：`python --version`
- 检查依赖是否安装：`pip install -r requirements.txt`
- 检查端口8502是否被占用

### Q2: API调用失败
- 检查 `.env` 文件中的API密钥是否正确
- 确认DeepSeek账户有足够的余额
- 检查网络连接

### Q3: 浏览器未自动打开
- 手动访问：http://localhost:8502
- 检查防火墙设置
- 尝试使用其他浏览器

### Q4: 数据未保存
- 检查 `experiment_log.csv` 文件权限
- 确保有写入权限
- 检查磁盘空间

## 注意事项

1. **实验伦理**：本平台为模拟研究，不涉及真实人类参与者
2. **数据安全**：API密钥等敏感信息存储在本地 `.env` 文件中
3. **资源消耗**：对话生成需要API调用，请注意使用频率
4. **学术用途**：本平台适用于心理学实验研究

## 技术支持

如有问题，请检查：

2. 检查 `experiment_log.csv` 文件
